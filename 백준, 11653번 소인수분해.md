## 🐍 나의 풀이 코드

```python
import sys

N = int(sys.stdin.readline().strip())

cnt = 2
while N > 1:
     if N%cnt == 0:
          N //= cnt
          print(cnt)
          cnt = 2
     else:
          cnt +=1
```
