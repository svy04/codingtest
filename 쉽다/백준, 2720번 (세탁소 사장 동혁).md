## 📝 문제 풀이 과정 및 간단 메모

> 나눈 값이 잔돈 단위의 개수일 것이고 나머지로 다시 나누면 될듯


## 🐍 나의 풀이 코드

```python
import sys

n = int(sys.stdin.readline().strip())

for _ in range(n):
     temp = [0 for _ in range(4)]
     i = int(sys.stdin.readline().strip())
     if i // 25 > 0:
          temp[0] += i//25
          i -= temp[0]*25
     if i//10:
          temp[1] += i//10
          i -= temp[1]*10
     if i//5:
          temp[2] += i//5
          i -= temp[2]*5
     temp[3] = i
     print(" ".join(list(map(str, temp))))

```
