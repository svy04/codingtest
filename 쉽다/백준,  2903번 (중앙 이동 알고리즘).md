## 📝 문제 풀이 과정 및 간단 메모

> 중앙에 한다는건 결국 한개가 부족함 2개 = 1개니까.
> 결론, 현재 상태 한번 더 돌리면 현재상태 + (현재상태-1)^2



## 🐍 나의 풀이 코드

```python
import sys

i = int(sys.stdin.readline().strip())

general = [1, 1]
if i == 0:
     print(len(general)**2)
else:
     for _ in range(i):
          for _ in range(len(general)-1):
               general.append(1)
     print(len(general)**2)

```

## 다른 풀이

N = int(input())
print((2**N+1)**2)
